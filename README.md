# Introduction to Algorithms

## Prerequisites

```bash
# WSL; install Rust + Cargo
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Cargo commands

### Build debug

`cargo build`

### Run

Will auto-build.

`cargo run`

### Build release

`cargo build --release`

### Test

`cargo test`
